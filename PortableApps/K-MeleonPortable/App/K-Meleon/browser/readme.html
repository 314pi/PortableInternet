Welcome to K-Meleon !

Getting Started

  • The K-Meleon home page is the place to visit to learn more about all aspects of the K-Meleon project.
  • Please read the Release Notes for important details about this version of K-Meleon.
  • We also have a FAQ that answers your Frequently Asked Question about K-Meleon. Start there if you have questions about K-Meleon.
  • More documentation is available including simple tutorials with examples and walk-throughs at the K-Meleon Documentation page and at the K-Meleon User's Guide and Reference Manual (still in progress).

Enhancing K-Meleon

  • In the Resource section you will find details on how to change K-Meleon's look or to use your favorite third-party plugin.

Need Help?

  • Our User Forums provide a place for users to discuss K-Meleon configurations and to help each other troubleshooting problems.

Get Involved

  • The K-Meleon project is a community driven project where users like you help each other to develop their browser of choice. Please consider contributing time and effort by getting involved in the project.
  • All known bugs are kept in our Bug Reporting System. If you find a bug, please report it there but first check with the FAQ and the User Forums to make sure it is a new bug or that you have new information to report.
