Bienvenue dans K-Meleon !

Débuter avec K-Meleon

  • La page d'accueil K-Meleon est l'endroit à consulter pour apprendre plus sur tous les aspects du projet K-Meleon.
  • Veuillez lire les notes de version concernant des détails importants de cette version de K-Meleon.
  • Vous avez aussi la Foire aux Questions (FAQ) qui répond aux questions fréquemment posées sur K-Meleon. Commencez par ici si vous vous posez des questions sur K-Meleon.
  • Une documentation importante est disponible incluant des tutoriaux simples avec des exemples renvoyant fréquemment sur la page de documentation de K-meleon et dans le Guide utilisateur de K-Meleon et Manuel de référence (en cours d'élaboration).

Améliorer K-Meleon

  • Sur la page des ressources vous pouvez trouver des détails sur les façons de personnaliser K-Meleon selon vos goûts ou d'utiliser vos outils ou plugins favoris.

Besoin d'aide?

  • Nos forums utilisateurs (en anglais) mettent à votre disposition un espace pour parler de la configuration de K-Meleon et pour vous aider à résoudre les problèmes que vous pourriez rencontrer avec K-Meleon.

Vous impliquer

  • Le projet K-Meleon est un projet de développement communautaire où des utilisateurs comme vous en aident d'autres à développer le navigateur de leur choix. Vous pouvez vous aussi contribuer à l'amélioration du projet en donnant de votre temps et de votre énergie (l'aide peut être dans tous les domaines: programmation, écriture de documentation, ...).
  • Tous les bogues connus sont référencés dans notre Système de report de bogues. Si vous trouvez un bogue, veuillez le reporter là, mais seulement après vous être assuré au préalable dans la Foire aux Questions, et dans les forums utilisateurs de la nouveauté du bogue ou afin d'apporter de nouveaux éléments à faire remonter.
