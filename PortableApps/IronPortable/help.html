Starting Iron Portable in Incognito mode

Copy IronPortable\Other\Source\AppNamePortable.ini to the IronPortable folder (the same folder as IronPortable.exe) and rename it to IronPortable.ini. Open the file in a text editor and change Incognito=false to Incognito=true, then restart Iron Portable. It should open in incognito mode.

Syncing Iron Portable with Google

Copy IronPortable\Other\Source\AppNamePortable.ini to the IronPortable folder (the same folder as IronPortable.exe) and rename it to IronPortable.ini. Open the file in a text editor and change AdditionalParameters= to AdditionalParameters=--user-agent="Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/38.0.2125.101 Safari/537.36", then restart Iron Portable. You should now be able to login to your Google account and enable sync. Once you do, close Iron, then remove the INI file. If you're using other options in the INI file, just reset AdditionalParameters= to be blank again.
