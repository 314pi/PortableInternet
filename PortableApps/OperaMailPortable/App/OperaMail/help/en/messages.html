Keyboard shortcuts

Messages

Messages are organized into "views", listed under their respective category heading. Press Tab to cycle through categories (and the message view). Use the up and down arrow keys to select views and press Enter to display the view's contents. Press j to navigate through messages inside a view and Enter on a highlighted message to display it in its own tab.

Writing messages

  • Click the “Compose” button in the Mail panel, click the "Compose" icon in the tab browser, or press Ctrl⌘ + Option + M to compose a new message. Opera Mail automatically saves your message as a draft and can be found in the “Drafts” view if you don't send it. If you decide not to keep it as a draft, just click “Discard Draft” at the top-left.
  • After importing accounts, Opera Mail offers to auto-complete email addresses from your contacts list.
  • To add files to the message, click “Attach”, or drag and drop files into the attachments field.
  • Opera Mail uses Rich Text Formatting (RTF) by default. If you prefer plain text, click the “Use Plain Text” button, located on the top-right of the window. To return to rich text, click the “Use HTML Formatting” button.
  • Opera Mail automatically checks your messages for spelling and errors, marked by a dotted red line. Right+Ctrl-click a marked word for a list of spelling suggestions.
  • To add a graphic smiley or emoticon, type :) or :(, or rightCtrl-click to display the context menu and select Insert > Smiley.
  • Send the message by clicking the “Send” button, or pressing Ctrl+Shift+S⌘+Enter.
  • Double-click a message in the Drafts view to continue editing a draft.
  • Double-click a message in the Sent view to edit and resend an old message.

Reading messages

New messages

  • Opera Mail checks and downloads your messages at regular intervals. If you wish to do this manually, for example if you ware using a dial-up connection, click the “Check/Send” button in the Mail panel or press Ctrl⌘+K.
  • Click the “Unread” view, under “All Messages”, to check new mail received to all of your accounts.
  • Under individual account category headers, most service providers place new mail in the Inbox folder. Click this folder to see new mail only delivered to a certain account.
  • Opera Mail highlights new mail in blue.
  • Messages that have been viewed in Opera Mail but are marked marked as unread will appear in bold.
  • Message threads are shown by default. To toggle threads, click the “Threads” button on the top-right of the messages list view. To view and set thread options, select the arrow next to the “Threads” button.

Read messages

  • Messages that are marked as read appear with normal text.
  • To mark a message as read, click the “Read” button on the top-left of the message view. See the Settings page to set the “Mark as Read” function automatically.
  • To quickly mark a message as “read” and remove it from the Unread view, highlight it and press K. The next unread message displays.
  • In the Received view, messages that have been marked as read are preceded by an empty circle rather than a dot.
  • The message list view marks mail you have replied to with an arrow.

Search messages

  • To find messages quickly, use the search field at the top of the messages list view. Opera Mail searches messages as you type.
  • To search for text contained in the message body, press Ctrl+F or F3⌘+F to toggle the text search bar.

Pin messages

To “pin” an important message, click the “Pin” button at the top-right of the message, or click the dot or arrow next to the message in the message list view. Pinned messages are shown under the “Pinned” category header, so you can find them quickly.

Delete messages

  • To delete a message, click the “Trash” button, located on the top bar of the message view.
  • When you delete a message from a view, it is not actually deleted, but moved to the “Trash” category.
  • To retrieve a deleted message, open the “Trash” view and click the "Undelete" button.
  • To delete a message permanently, click the arrow next to the “Trash” button and select “Delete Permanently” or press Shift+DelCtrl-click > Delete > Delete Permanently.
  • To delete all messages in Trash, right-click inside the Mail panel and select “Empty trash”.

Sort messages

To sort listed messages, click the “Settings for this view” button at the top-right of the message list and choose an option.

Responding to messages

To respond to a message you received, you can use the message view toolbar to reply, forward, mark as read or spam, delete or label the message. Alternatively, you can rightCtrl-click the message in the message list view for more options.

“Quick Reply” helps you easily reply to messages by placing a message reply window directly below the message. To enable Quick Reply, click the “Default Mail Settings” button at the top-right of the message display and check the “Show Quick Reply” check box.

Customizing the Compose Message view

You can change the settings for the compose window to your preferred look and feel. By clicking the “Default Mail Settings” button at the top-right, you can choose the following options:

  • “Maximize Messages” sets the compose window to fit the full window width.
  • The “Show Fields” check-boxes set the mail fields that are displayed in the compose window.
  • The “Edit Signature...” button allows you to change your signature on outgoing messages. Clicking this button will open the “Signature Editor” window, which features an option to change your signature for all outgoing accounts.
  • The “Default outgoing account” drop-down menu sets which account the current message will be sent through.
