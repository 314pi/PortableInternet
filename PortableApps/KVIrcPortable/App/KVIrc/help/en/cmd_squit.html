squit

Disconnects a server socket (ircops only)
Usage 
squit <server> [:]<comment>
Description
Disconnects a server socket: this is for IRC operators only.
This command is a rfc2821 command wrapper; see that document for more information.
WARNING: the <comment> parameter may contain spaces: in that case you MUST add the leading ':' character before.
