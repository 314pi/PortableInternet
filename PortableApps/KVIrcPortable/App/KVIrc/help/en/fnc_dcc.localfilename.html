$dcc.localFileName

Returns the local file name associated with the specified DCC
Usage 
$dcc.localFileName(<dcc_id>)
Description
Returns the local file name associated with the specified DCC.
If <dcc_id> does not identify a file transfer DCC then this function returns an empty string. If <dcc_id> is not a valid Direct Client Connection identifier then this function prints a warning and returns an empty string.
Syntax Specification 
<string> $dcc.localFileName(<dcc_id:uint>)
