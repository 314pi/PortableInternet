$k

Returns the COLOR mIRC control character
Usage 
$k(<foreground>[,<background>])
$k
Description
Returns the COLOR mIRC control character (Qt::CTRL+K).
If <foreground> and <background> are passed, a standard mIRC color escape is returned.
