$mediaplayer.mediaType

Returns the currently played media type
Usage 
$mediaplayer.mediaType()
Description
Returns a string describing the media type currently played by the selected media player interface. This function is not granted to work with all the player interfaces and all the media types. Take a look at the mediaplayer module documentation for more details about how it works.
Syntax Specification 
$mediaplayer.mediaType()
