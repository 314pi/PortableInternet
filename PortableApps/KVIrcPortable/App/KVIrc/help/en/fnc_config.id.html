$config.id

Returns the id of an open config file
Usage 
$config.id(<filename>)
Description
Returns the id of an open config file specified by <filename>.
(This is the same id that is returned by $config.open.
If no such file is open, 0 is returned (that is an invalid config identifier).
Syntax Specification 
$config.id(<filename>)
